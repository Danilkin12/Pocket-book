 Библиотека PDF-книг

Главная страница

Веб-приложение для хранения и обмена PDF-книгами с системой рейтингов и комментариев.

  Возможности

-  Каталог PDF-книг с метаданными
-  Поиск по названию и автору
-  Система оценок 
-  Комментарии к книгам
-  Ролевая модель 
-  Загрузка и скачивание книг
-  Аутентификация и авторизация

 Технологии

Backend:
- Python 3.8<
- Flask 2.0
- Flask-Login
- SQLite 3

Frontend:
- HTML5, CSS3, JavaScript
- Jinja2 
- Font Awesome 

Инфраструктура:
- Werkzeug  

 Установка и запуск

 Требования
- Python 3.8<
- pip
